# Responses & Follow-ups

## The Core Principle

Every email deserves a response. Not every email deserves an answer.

Your goal: respond in a way that looks helpful, buys time, and transfers ownership.

---

## The Acknowledgement Email

When you have no idea what to do but need to appear responsive.

```
Subject: Re: [Original Subject]

Hi [Name],

Thanks for bringing this to my attention — really appreciate you
flagging it. I want to make sure I'm giving this the proper thought
it deserves before responding, so let me take a look and come back
to you shortly.

In the meantime, do you have any additional context that might
be helpful?

Best,
[You]
```

**Effect**: Buys 5-10 business days. "Shortly" is not legally defined.

---

## The Soft No

Declining without using the word "no."

```
Subject: Re: [Request]

Hi [Name],

Thanks so much for thinking of me for this — really glad you reached out.

I want to be thoughtful here. Given current priorities and bandwidth
on my end, I want to make sure this gets the attention it deserves.
I'd hate to commit and then underdeliver.

It might be worth exploring whether [other person/team] could be
better positioned to support here — they've been doing great work
in this space.

Happy to connect and help think through the best path forward
if that would be useful.

Best,
[You]
```

**What was said**: No.
**What it reads as**: Thoughtful, helpful, strategic.

---

## The Gracious Acceptance (With Caveats)

Agreeing to do something while immediately limiting what you'll actually do.

```
Subject: Re: [Request]

Hi [Name],

Absolutely — happy to help with this.

A few things I want to flag upfront so we're aligned:
- I'll need [dependency] from [other team] before I can proceed
- Timeline will depend on current sprint commitments
- I'll plan to share an initial version for feedback before finalizing

Does that work? Let me know if you'd like to sync to align on scope first.

Thanks,
[You]
```

**Effect**: You agreed. You have also installed 3 blockers that will delay indefinitely.

## The "Per My Last Email"

For when they ignored you the first time.

```
Subject: Re: Re: [Original Subject]

Hi [Name],

Circling back on the below — just wanted to make sure this didn't
get lost in the shuffle.

Per my last email, [restate the exact thing you said before verbatim].

Happy to jump on a quick call to align if that's easier.

Best,
[You]
```

**Tone**: Polite.
**Subtext**: I have a timestamp. I have receipts. Do not test me.

---

## The Follow-Up Ladder

Use this escalating sequence when someone keeps ignoring you.

### Follow-up #1 (Day 3)
```
Hi [Name], just wanted to make sure this didn't get buried — following
up on the below. Let me know if you have any questions!
```

### Follow-up #2 (Day 7)
```
Hi [Name], circling back one more time on this. Happy to jump on a
quick call if that's easier than email.
```

### Follow-up #3 (Day 12)
```
Hi [Name], I want to make sure we're not blocked on this — looping
in [their manager] just to keep visibility on the timeline.
```

**Note**: Follow-up #3 is the nuclear option. Use it. It works every time.

## The Handoff

Transferring full ownership to someone else while appearing to stay involved.

```
Subject: Intro: [Name A] <> [Name B] — [Vague Topic]

Hi both,

[Name A], wanted to connect you with [Name B] — [Name B] is the
right person to help move this forward and has great context on
[topic].

[Name B], [Name A] is looking to [vague description of what they want].
I'll let you two connect directly from here — happy to rejoin the
thread if helpful.

Thanks!
[You]
```

**Effect**: You are now CC'd but no longer responsible. You can observe from safety.
